GPIO CLIENT EXTENSION USING ONC RPC

This is a Microsoft Small Basic extension for communicating with
a Munts OS GPIO Thin Server, using ONC RPC.  Visit the following
pages for more information on the server:

https://github.com/pmunts/muntsos/extensions/GPIO
https://github.com/pmunts/muntsos/thinservers/GPIO
