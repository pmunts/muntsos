Embedded Linux Application Libraries

This directory contains a recipe for building some development libraries for a particular MuntsOS target.

The following libraries are included:

  • GNU ncurses library
  • libusb USB library
  • OpenSSL crypto library
  • zlib compression library
  • libcurl HTTP access library
  • xmlrpc-c XML RPC library
  • MySQL database client library
  • libpcap packet capture libarary
  • Netlink protocol library
  • RabbitMQ AMQP client library
  • libsodium crypto library
  • ZeroMQ messaging library
  • gdbm database library

Installation

Build the libraries with:

make BOARDNAME=RaspberryPi1

Build installation packages with either of:

make BOARDNAME=RaspberryPi1 package.deb

make BOARDNAME=RaspberryPi1 tarball

Prebuilt toolchain packages are available at: https://repo.munts.com.
